About This Project

This Polynomial Calculator allows you to solve polynomial equations by inputting coefficients and variables (x, y, z). It’s built using the following technologies:

  • HTML: Structure and content.
  • CSS: Styling and layout.
  • JavaScript: Logic and interactivity.

Created to help users quickly compute polynomial values using multiple variables.
